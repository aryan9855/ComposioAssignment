# Composio App Research Agent

This workspace contains a lightweight case-study deliverable for the 100-app audit.

## Files
- `index.html` — self-contained HTML case study page.
- `research_agent.py` — deterministic research-agent style script that clusters the app list and emits a JSON summary.
- `research_output.json` — generated by the script.

## Run
```bash
python research_agent.py
python -m http.server 8000
```
Then open `http://localhost:8000` in a browser.

## Notes
This is intentionally transparent about what it can infer directly and where human verification is still required.
